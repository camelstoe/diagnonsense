# Project Diagnonsense

A free, open-source symptom tracker built for people navigating undiagnosed or early-stage joint inflammatory conditions.

Tracks joint temperatures (with automatic differentials against control sites), circumference measurements, bilateral pain scores with type detail, headache patterns, sun exposure correlation, medications, and photos flagging — all synced to a personal Google Sheet.

Built with plain HTML/JS. No account needed. Works on any device.
